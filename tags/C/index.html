模板类型推断

  • 当以引用的方式进行参数传递，实例化时必须保持类型匹配
  • 当以值的方式进行参数传递，只支持type decay
    • 忽略identifer qualification：const, volatile等
    • 引用被转换为相应的引用的类型
    • raw array或函数被转换成相应的指针

函数模板重载

  • C++支持函数模板和普通函数的重载，在调用时如果类型匹配，会优先调用非模板函数
  • 由于模板函数不考虑自动类型转换，在调用时遇到需要转换，会选择调用非模板函数
  • 在重载函数模板时，应当避免出现歧义，即同一个template instantiation同时match同个template declaration
